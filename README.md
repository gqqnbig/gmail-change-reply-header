# gmail-change-reply-header

## 安装

创建一个书签，其地址为
```
javascript:fetch("https://raw.githubusercontent.com/gqqnbig/gmail-change-reply-header/master/fix.js").then(response => { response.text().then(function (text) {eval(text)}); })
```
设置回复行为英文。

要用时点击该书签即可。
